# 模块与包

## 模块

一个 `.py`结尾的源文件即为一个模块。其中可以包含变量、函数和类等

## 创建模块

模块名区分大小写，不能与Python自带模块重名

`my_add.y`

```py
num = 100
def add(a,b):
    return a+b
```

## 导入模块

### 全部导入

导入模块的所有成员，通过`模块名.成员名`的方法访问

```py
import 模块名 [as 别名]
```

`10_模块与包.py`

```py
import my_add as a1

print(a1.num)
print(a1.add(1,2))
```

### 局部导入 from import

导入模块的部分成员，直接通过`成员名`的方式访问

**如果导入的成员有重名，后导入的成员会覆盖前面导入的成员**

```py
from 模块名 import 成员名1[as 别名],成员名2[as 别名]
```

```py
from my_add import num
print(num)
print(add(1,2)) #未导入的成员不能用
```

### 局部导入2

导入模块中所有**不以单下划线开头**的成员，直接通过`成员名`访问

```py
from 模块名 import *
```

```py
from my_add import *
print(num)
print(add(1,2))
```

### 模块搜索顺序

当导入模块时会按照

1. 当前目录

2. PYTHONPATH环境变量中的目录

3. Python 安装的第三方库目录(site-packages,pip安装包存放位置)

4. 标准库目录（Python 内置标准模块：os、sys、math 等）

5. Python解释器自带的默认库目录(编译时内置路径)

可以通过下面的代码查看 

```py
import sys
print(sys.path)
```

也可以通过`sys.path.append(路径)`向sys.path中临时添加路径

```py
import sys

sys.path.append("./..")
print(sys.path)
```

### `__all__`

使用**from import ***导入模块时，可以在**被导入的模块中**使用`__all__`设置哪些内容可以被导入

`my_add.py`

```py
__all__ = ["num"]

num = 100
num1 =50
```

```py
from my_add import *
print(num)
print(num1) #NameError: name 'num1' is not defined.
```

### `__name__`

`__name__`是一个特殊的内置变量

① 脚本直接运行时

```py
print(__name__)
# 输出：__main__
```

② 文件被 `import` 导入时

在`a.py`:

```py
print(__name__)
```

在 `b.py` 执行 `import a`，输出：

```textile
a
```

值为**当前模块文件名**（不含 `.py`）

**经典场景**

```py
def test():
    print("函数执行")

if __name__ == '__main__':
    test()
```

只有直接运行本文件的时候才会执行

## dir()

`dir()`是一个内置函数，主要用于列出对象的属性和方法，或者列出当前作用域中定义的名称，并以一个字符串列表的形式返回

1. 以一个模块作为dir()的参数
-  返回模块中定义的名称列表，包括函数，类，变量等

```py
import math
print(dir(math))
```

2. 以一个对象作为dir()的参数
- 返回这个对象的属性和方法列表

```py
str1 = str()
print(dir(str1))
```

3. 以一个类作为dir()的参数

```py
print(dir(str))
```

4. 不带参数：返回当前作用域内的名称列表；



## 包概述

   包是多个模块的文件夹集合，通常将多个有联系的模块放入一个包中

   包和文件夹类似，但是这个文件夹下必须有一个`__init__.py`文件  

   帮助我们更清晰的管理模块

    mypackage/          # 根包
        __init__.py
        math/            # 子包
            __init__.py
            calc.py     # 子模块
        string.py

## 导入包

### 全局导入`import`

语法

- 通过 `包名.模块名` 或者 `别名`访问 

```py
import 包名.模块名 [as 别名]
```

```py
import graphic.circle as c
print(c.PI)
```

在使用`import`时，除最后一项外必须是**包**。最后一项可以是包或模块，但不能是类、函数或者变量

如果最后一项是**包**那么必须在包的`__init__.py`中，指定导入包的哪些模块，指定导入模块的时候建议使用**相对路径**

这是python导包的一个优化机制，避免导入过多模块

`__init__.py`

```py
from . import circle
from . import rectangle
```
