# 运算符
## 算数运算符
```py
a,b=10,3
print(a+b,'+ 加')
print(a-b,'- 减')
print(a*b,'* 乘')
print(a/b,'/ 除')
print(a//b,'// 取整')
print(a%b,'% 取余')
print(a**b,'** 幂')
```

## 比较运算符
- 返回结果是bool类型
- 不同的数据类型之间不能大小比较
- 


```py
num1=10
num2=20
print(num1==num2)
print(num1!=num2)
print(num1>num2)
print(num1<num2)
print(num1>=num2)
print(num1<=num2)
```

## 逻辑运算符
- and 与，x and y 如果 x 为 False则返回x的值，否则返回y的值
- or 或，x or y若x 为 True，则返回x的值，否则返回y的值
- not 非，not x 若x为Ture则返回False,x为False则返回True
