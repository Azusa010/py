# 04_容器数据类型
## 有序
- 放进去的顺序和取出的顺序一致
- 每个元素都有对应的索引
- 0 -> 递增
- 递减 <- -1

## 列表
- 可变，有序，可以重复
- 列表元素可以是不同类型
```py
list1 = [10,'hello',20.5,True]
```

- 列表切片
```py
list1 = [10,'hello',20.5,True,100]
#取全部列表
print(list1)
#复制整个列表
print(list1[:])
#取索引从2开始到4(不包括)的元素
print(list1[2:4])
#取索从2到末尾的元素
print(list1[2:])
#从0到2
print(list1[:2])
#从2到-1
print(list1[2:-1])
#倒取元素
print(list1[::-1])
```

- 添加元素
```py
list1.append(150)
list1.insert(index,object)
```


- 列表相加
```py
list1=[1,2,3]
list2=['a',False,'b']
print(list1+list2)
```

- 列表乘法
```py
list1 = [1, 2, 3]
print(list1 * 3)
#[1, 2, 3, 1, 2, 3, 1, 2, 3]
```

- 修改列表中的元素
```py
#1
list1[2]=600

#切片
list1 = [100, 200, 300,400,500]
list1[1:2] = [700,800]
print(list1)

#常见操作
len(list1),max(list1),min(list1),sum(list1)

```

- 列表遍历
```py
#1)
for item in list1:
    print(item,end='\t')
#2)
for i in range(len(list1)):
    print(list1[i])

#3)
for index,item in enumerate(list1):
    print(index,item)

```

- 元素删除
```py
del list1[0]
print(list1)

list1.remove(200)
```

- 列表嵌套
列表中元素可以为列表

- 列表推导式
```py
list2 = [item for item in list1 if item!=200]


#2)
list1=[1,2,3,4,5]
list2=['a','b','c','d','e']
list3 = [(i,j) for i in list1 for j in list2]
print(list3)
```

- zip()函数
将多个可迭代对象打包为一个个元组
```py
list1=[1,2,3,4,5]
list2=['a','b','c','d','e']
zipped = zip(list1,list2)
print(list(zipped))
#[(1, 'a'), (2, 'b'), (3, 'c'), (4, 'd'), (5, 'e')]
```
- 列表常用方法
```py
list1=[1,2,3]
list2=[4,5,6]
list1.extend(list2) #在list1上追加


list1.pop([index])#删除指定索引的元素
list1.clear() #清空列表中元素
list1.sort([reverse=]) #排序
list1.reverse() #反转列表
list1.index(x[,start,stop]) #返回x在列表内第一次出现的索引
list1.count(x) #返回x的数量
list1.copy() #拷贝列表
list(x) #将序列转换为列表
```

## 字符串
- 不可变，有序
```py
str1="""
"""
#三引号，保留格式
```
- 加法
- 乘法
- 检查成员是否为字符串元素
```py
str1 = 'hello world'
print('ll' in str1) #True
```
- 原始字符串
```py
print(r'hello \n world')
```
- 常用函数
```py
str1 = 'abcdefcccc'
print(str1.replace('c','a')) #replace(old,new[,max])

#.split([x][,n]) 按x分割字符串，可设置最大分割次数
str1='a,b,c,d,e,f,g,h,i'
print(str1.split(',')) #['a', 'b', 'c', 'd', 'e', 'f', 'g', 'h', 'i']


#rsplit 从右往左
str1 = "a,b,c,d,e,f,g,h,i"
print(str1.rsplit(",", 2))

#x.join(seq)  使用x把列表中的元素连接合成新的字符串
print('_'.join(["a","b","c"])) #a_b_c

#strip([x]) 裁掉字符串两边的空格或指定字符
#lstrip , rstrip

#removeprefix/removesuffix 裁掉字符的前缀/后缀
str1='hello'
str2=str1.removeprefix('he')

#将全部字符转换为大写/小写
#upper()/lower()
str1='hello'
str2=str1.upper()

#swapcase()反转大小写
str1='HEllo'
str2 = str1.swapcase()

#capitalize第一个字母变成大写，其他的变成小写
str1='hEllo'
str2 = str1.capitalize()

#casefold 将字符串彻底转换为小写，甚至能处理特殊字符
str1='hEllo'
str2 = str1.casefold()

#返回字符串第一个x出现的索引值,不存在返回-1
str1='hEllo'
str2 = str1.find('h')

#rfind(x)从右往左找
str1='hElloh'
str2 = str1.rfind('h')

#index(x[,start][,end]) 与index()类似,不存在报错
str1='hElloh'
str2 = str1.index('h')

#count(x[,start][,end]) 返回x的个数
str1='hElloh'
str2 = str1.count('h')

#startswith(x[,start][,end]) 检查是否x开头
str1='hElloh'
str2 = str1.startswith('h')

#endswith

#isspace()判断字符串是否非空且只包含空白

str.isalnum()#检查是否非空，且只包含字母和数字
str.isalpha()#检查是否非空且只包含字母
str.isascii()#检查是否只包含ASCII字符,包括空字符串
str.isdecimal()#检查是否非空且只包含十进制字符
str.isdigit()#检查是否非空且只包含数字
str.isidentifier()#检查是否有效的标识符
```

## 元组
- 有序，不可变，元素可以重复
- 通过()定义
- 通过推导式创建元组对象 
```py
tuple1=(i for i in range(5))  #创建的是生成器对象
print(type(tuple1)) #<class 'generator'>
print(tuple(tuple1)) #(0, 1, 2, 3, 4)
```
- 访问元组(切片)
- 元组相加
```py
tuple1=tuple((i for i in range(5)))
tuple2=(1,2,3)
print(tuple1+tuple2) #(0, 1, 2, 3, 4, 1, 2, 3)
```
- 元组乘法
```py
tuple2=(1,2,3)
print(tuple2*2) #(1, 2, 3, 1, 2, 3)
```
- 用 in 判断元素是否在元组中
- len(),max(),min(),sum()
- 遍历：同列表

## 集合
- 无序,不包含重复元素，可以是不同类型
- 使用{}定义，用，分割，也可以用set()定义
- 没有索引，不可切片
