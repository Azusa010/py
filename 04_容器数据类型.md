# 04_容器数据类型
## 有序
- 放进去的顺序和取出的顺序一致
- 每个元素都有对应的索引
- 0 -> 递增
- 递减 <- -1

## 列表
- 可变，有序，可以重复
- 列表元素可以是不同类型
```py
list1 = [10,'hello',20.5,True]
```

- 列表切片
```py
list1 = [10,'hello',20.5,True,100]
#取全部列表
print(list1)
#复制整个列表
print(list1[:])
#取索引从2开始到4(不包括)的元素
print(list1[2:4])
#取索从2到末尾的元素
print(list1[2:])
#从0到2
print(list1[:2])
#从2到-1
print(list1[2:-1])
#倒取元素
print(list1[::-1])
```

- 添加元素
```py
list1.append(150)
list1.insert(index,object)
```


- 列表相加
```py
list1=[1,2,3]
list2=['a',False,'b']
print(list1+list2)
```

- 列表乘法
```py
list1 = [1, 2, 3]
print(list1 * 3)
#[1, 2, 3, 1, 2, 3, 1, 2, 3]
```

- 修改列表中的元素
```py
#1
list1[2]=600

#切片
list1 = [100, 200, 300,400,500]
list1[1:2] = [700,800]
print(list1)

#常见操作
len(list1),max(list1),min(list1),sum(list1)

```

- 列表遍历
```py
#1)
for item in list1:
    print(item,end='\t')
#2)
for i in range(len(list1)):
    print(list1[i])

#3)
for index,item in enumerate(list1):
    print(index,item)

```

- 元素删除
```py
del list1[0]
print(list1)

list1.remove(200)
```

- 列表嵌套
列表中元素可以为列表

- 列表推导式
```py
list2 = [item for item in list1 if item!=200]


#2)
list1=[1,2,3,4,5]
list2=['a','b','c','d','e']
list3 = [(i,j) for i in list1 for j in list2]
print(list3)
```

- zip()函数
将多个可迭代对象打包为一个个元组
```py
list1=[1,2,3,4,5]
list2=['a','b','c','d','e']
zipped = zip(list1,list2)
print(list(zipped))
#[(1, 'a'), (2, 'b'), (3, 'c'), (4, 'd'), (5, 'e')]
```
- 列表常用方法
```py
list1=[1,2,3]
list2=[4,5,6]
list1.extend(list2) #在list1上追加


list1.pop([index])#删除指定索引的元素
list1.clear() #清空列表中元素
list1.sort([reverse=]) #排序
list1.reverse() #反转列表
list1.index(x[,start,stop]) #返回x在列表内第一次出现的索引
list1.count(x) #返回x的数量
list1.copy() #拷贝列表
list(x) #将序列转换为列表
```
