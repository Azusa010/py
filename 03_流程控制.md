## 分支
- 单分支 if
```py
balance = randint(0,100)
print(f'当前余额为{balance}')
price = 50
if balance < price:
    print(f'余额不足{price},请充值')
else:
    print('欢迎下次光临')
```

- 双分支  if else
```py
balance = randint(0,100)
price = 50
print(f'当前余额为{balance}')
if balance < price:
    print(f'余额不足{price},请充值')
else:
    balance-=price
    print(f'消费成功，剩余余额为{balance}')
print('欢迎下次光临')
```
- 多分支  if elif else
```py
age = randint(0,100)
print(f'这个人年龄是{age}岁',end='\t')
if age<2:
    print('婴儿')
elif age < 4:
    print('幼儿')
elif age < 13:
    print('儿童')
elif age < 20:
    print('青少年')
elif age < 65:
    print('成年人')
else:
    print('老年人')

```

- 嵌套分支  分支中又包含了分支
```py
# 嵌套分支
state = 0b000
if state & 0b111 == 0b111:
    print('大写状态')
else:
    if state & 0b010 == 0b010:
        if state & 0b001 == 0b001:
            print('简体中文-微软拼音')
        else:
            print('英文模式')
    else:
        print('英语美式键盘')
    
```

- match case语句

```py
match x:

```
