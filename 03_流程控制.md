# 流程控制
## 顺序
- 按照正常顺序，依次执行每条语句

## 分支
- 单分支 if
```py
balance = randint(0,100)
print(f'当前余额为{balance}')
price = 50
if balance < price:
    print(f'余额不足{price},请充值')
else:
    print('欢迎下次光临')
```

- 双分支  if else
```py
balance = randint(0,100)
price = 50
print(f'当前余额为{balance}')
if balance < price:
    print(f'余额不足{price},请充值')
else:
    balance-=price
    print(f'消费成功，剩余余额为{balance}')
print('欢迎下次光临')
```
- 多分支  if elif else
```py
age = randint(0,100)
print(f'这个人年龄是{age}岁',end='\t')
if age<2:
    print('婴儿')
elif age < 4:
    print('幼儿')
elif age < 13:
    print('儿童')
elif age < 20:
    print('青少年')
elif age < 65:
    print('成年人')
else:
    print('老年人')

```

- 嵌套分支  分支中又包含了分支
```py
# 嵌套分支
state = 0b000
if state & 0b111 == 0b111:
    print('大写状态')
else:
    if state & 0b010 == 0b010:
        if state & 0b001 == 0b001:
            print('简体中文-微软拼音')
        else:
            print('英文模式')
    else:
        print('英语美式键盘')
    
```

## 循环
- while循环  ==不明确循环的对象或者次数==
```py
while 表达式
    语句-循环体

#示例
week = 1
rabbits = 2
while week<10:
    rabbits= rabbits+rabbits*2
    week+=1
print(f'第{week}周有{rabbits}只兔子')
```

- while else
通常和break一起用，循环通过break终止后，else中的语句不会执行

- for  ==明确循环对象/循环次数==
```py
for 临时变量 in 可迭代对象:
    循环体
else:
    循环正常结束执行

#1)遍历列表
for i in [2,3,4,5,6,7,8,9,0]:
    print(i,end=' ')

#2)遍历字符串
for i in 'string':
    print(i)

#3)遍历range数列
for i in range(10):
    print(i)
    #range([start,]stop[,step])生成数列，返回一个可迭代对象,[) step步长
```

- continue
    跳出当前正在进行的循环，继续进行下一次循环
- break 
    结束循环
- pass
   占位
