# 文件操作
## 文件的打开与关闭
1. 打开文件
    使用open(文件名,模式)打开或创建文件，执行完毕后返回file对象
    <table><tr><th>模式</th><th>说明</th></tr><tr><td>r</td><td>读写方式：只读，文件若不存在会报错。默认此模式</td></tr><tr><td>w</td><td>读写方式：写入，写入前清空原有数据。文件不存在会创建文件</td></tr><tr><td>a</td><td>读写方式：追加写入，在原有数据后追加，文件不存在会创建文件</td></tr><tr><td>x</td><td>读写方式：创建新文件并写入，文件若已存在会报错</td></tr><tr><td>b</td><td>编码方式：以二进制打开。一般用于非文本文件如图片等</td></tr><tr><td>t</td><td>编码方式：以文本模式打开，默认此模式</td></tr><tr><td>+</td><td>能读能写</td></tr></table>
```py
test = open('test.txt','w')

#写入数据
test.write('hello,python\n')

# 关闭连接
test.close()
```

2. 读文件
```py
test = open('test.txt','rt')

#读取数据
#read() 无参数读取全部内容
#read(size) 读取 size 个字符/字节数
print(test.read(3))
print(test.read())

#readline([size]) 读取一行
#readlines([size]) 读取所有行并返回 列表

# 关闭连接
test.close()
```

3. 案例
```py
def copyPic(src, dst):
    pic = open(src, "rb")
    pic_copy = open(dst, "wb")

    while content := pic.read(1024):
        pic_copy.write(content)

    pic.close()
    pic_copy.close()

copyPic("pic.png", "pic_copy.png")

```
