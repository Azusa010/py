## `__init__`方法

__init__() 方法的主要作用是在创建类的对象时，对对象的属性进行初始化。

`__new__()`创建新的对象，并且把新的对象传递给`__init__()`方法作为第一个参数(通常为self)传递给他

`__init()`不是必须的，如果类中没有定义，默认会有空参构造函数

## self

### self作为实例传参

self代表类的自身实例

### 通过self在类中调用类的实例属性和实例方法

```py
class Person:
    """人的类"""

    home = "earth"

    def __init__(self, name):
        self.name = name

    def eat(self):
        print("eating...")

    def drink(self):
        print("drinking...")

    def eat_and_drink(self):
        print(self.name)  # 在类中调用name
        self.eat()  # 在类中调用eat()方法
        self.drink()  # 在类中调用drink()方法

p = Person("张三")  # 创建一个对象
p.eat_and_drink()
```

## 属性

### 类属性

#### 1.通过`类名.属性名`或`实例名.属性名`访问

```py
class Person:
    """人的类"""

    home = "earth"  # 定义类属性

print(Person.home)  # 通过类名访问类属性

p1 = Person()  # 创建一个实例对象
print(p1.home)  # 通过实例名访问类属性，(如果实例没有覆盖这个类属性的值)
```

#### 通过`类名.属性名`添加与修改类属性

```py
class Person:
    """人的类"""

Person.home = "earth"  # 添加类属性
print(Person.home)  # earth

Person.home = "mars"  # 修改类属性
print(Person.home)  # mars
```

不建议实例名与类名同名

## 方法

有三类方法：实例方法，静态方法，类方法

### 实例方法

实例方法在类中定义，第一个参数为self，代表实例本身

实例方法智能被实例对象调用

可以访问实例属性，类属性，类方法

```py
class Person:
    '''人类'''
    home = 'earth'

    def __init__(self,name,age):
        #实例属性
        self.name = name
        self.age = age

    def m2(self):
        print('m2方法被调用了')

    def m1(self):
        print(Person.home) 
        print(self.name)
        self.m2()

p1 = Person('张三',18)
p1.m1()
```

### 类方法

在类中通过`@classmethod`定义,第一个参数`cls`代表类本身

可以被类和实例调用

可以访问类属性

```py
class Person:
    '''人类'''
    home = 'earth'

    def __init__(self,name,age):
        #实例属性
        self.name = name
        self.age = age

    def m1(self):
        print(Person.home) 
        print(self.name)

    @classmethod
    def m3(cls):
        print(cls.home)
        print(cls.__doc__)

Person.m3() # earth 人类
```

### 静态方法

通过在类中`@staticmethod`定义

一般工具类方法定义为静态方法

```py
class Person:
    '''人类'''
    home = 'earth'

    @staticmethod
    def m4():
        print('人类')

Person.m4() # 人类
```

### 在类外定义方法

```py
def m5(self,x,y):

class Person:

Person().f1(1,2) # 3
```

## 动态添加属性和方法

给对象动态添加实例属性

```py
class Person:
    def __init__(self,name):
        self.name = name
        
p = Person('zs')
print(p.name)

p.age = 18
print(p.age)
```

动态给类添加类属性

```py
class Person:
    def __init__(self,name):
        self.name = name
        
p = Person('zs')
print(p.name)
Person.home = 'earth'
print(p.home)
```

给实例添加普通方法

```py
class Person:
    def __init__(self,name):
        self.name = name


def eat():
    print('eating')
p = Person('zs')
p.eat = eat
p.eat()
```

给实例添加实例方法

```py
import types
class Person:
    def __init__(self,name):
        self.name = name


def eat(self):
    print('eating')
p = Person('zs')
p.eat = types.MethodType(eat,p)
p.eat()
#p.eat = eat
#p.eat(p)
```

动态给类添加方法

```py
class Person:
    home = 'earth'

    def __init__(self,name):
        self.name = name
@classmethod
def come_from(cls):
    print(f'来自{cls.home}')
@staticmethod
def static_func():
    print('static function')
Person.come_from = come_from
Person.static_func = static_func
Person.come_from()
Person.static_func()
```

动态删除属性与方法

del 对象.属性名

delattr(对象,属性名)



`__slots__`限制实例属性与实例方法

只允许添加`__slots__`中声明的实例属性与实例方法
