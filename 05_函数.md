# 函数
## 概念
- 带名字的代码块，用于完成具体任务，可重复使用
- 提高代码复用性

- 定义的语法
```py
def 函数名(参数列表):
    函数完成的功能(函数体)
    [return 返回]
```

- 函数必须先定义再调用

### 参数
- 形式参数:没有分配存储空间也没有值
- 实际参数:将实参赋值给形参
- 对于可变数据类型不同的赋值方式的区别
    var = var * 2 创建新的对象
    var *= 2 不会创建新对象

- 函数可使用的参数形式
  1. 必须参数
  2. 关键字参数
  3. 默认值参数 : 非默认参数一定放在默认参数之前
  4. 不定长参数 :
     1. 用 *var_args_tuple 进行声明
     2. 用 **var 声明，放到字典中，后面不能有其他参数
     3. 如果不定长参数后面还有参数，用关键字参数传参
     4. 必须参数在关键字参数之前
```py
def print_info(a,b=2,c=3):
    print(a,b,c)

print_info(1,2,3)
print_info(a=1,b=2,c=3)
print_info(1)

#不定长参数
def prt(a,*args,c):
    print(a,args,c) 
prt(1,2,3,4,5,c=1) #    1 (2, 3, 4, 5) 1

def pr(a,**args):
    print(a,args)
pr(1,b=1,c=2,d=3) #     1 {'b': 1, 'c': 2, 'd': 3}
```

- 解包传参
```py
def func(a,b,c):
    print(a,b,c)
    
tup1=(1,2,3)
func(*tup1)
dict1={'a':1,'b':2,'c':3}
func(**dict1) #dict的key和func的形参要一致
```

- 强制使用位置传参和关键字传参
    1. / 前必须用位置传参  
    2. *后必须用关键字传参
```py
def func(a,b,/,c,*,d,e):
    print(a,b,c,d,e)
func(1,2,3,d=1,e=2)
```

- 深拷贝 : 拷贝后所有层都是独立的


### 作用域
- 闭包
    1. 在一个函数体中定义另一个函数
    2. 外层函数返回值是内层函数对象
    3. 内函数中访问了外函数中定义的变量
   
   延长外层函数中定义的变量的生命周期，内层函数执行完才会销毁

- 作用域
    - 局部Local -> 嵌套Enclosing -> 全局Global -> 内嵌Built-in


- 用 global 声明全局变量,可以修改全局变量
```py
var1 = 10
def func():
    global var1
    var1 = 20
    print(var1)
func()
print(var1)
```

- 修改可变类型的全局变量
  
```py
list1 = [1,2,3]
def func():
    list1[1]=20
    print(list1)
func()
print(list1)
```

```py
def func():
    var1=10
    def f():
        nonlocal var1  #声明嵌套变量
        var1 = 20
        print(var1)
    f()
    print(var1)
func()
```

### 递归
```py
def get_factorial(n):
    if n==1:
        return 1
    return n * get_factorial(n-1)

print(get_factorial(5))
```

### 匿名函数
- lambda 参数列表:表达式
```py
def calculate(num1,num2,op):
    return op(num1,num2)

calculate(10,20,lambda a,b : a+b)
```
```py
list1 = [i for i in range(5)]
print(list(map(lambda item:'py'+str(item),list1)))
```
```py
list1 = [i for i in range(10)]
print(list(filter(lambda i: i%2==0,list1)))
```
```py
from functools import reduce
print(reduce(lambda x,y:x+y,[1,2,3,4,5]))
```
