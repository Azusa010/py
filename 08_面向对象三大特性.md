# 三大特性

## 封装

将变量和函数写入类中的操作 即为封装，类中封装了属性和方法

通过封装，可以将一些细节隐藏起来，之暴露出必要的接口给使用者使用

### 私有化

#### 1.单下划线：非公开API

不具备强制力

#### 2.双下划线，名称改写

至多一个下划线结束

只能通过`_类名__x`访问

### 私有属性

```py
class Person:
    def __init__(self,name,age):
        self.name = name 
        self.__age = age

p1 = Person('张三',20)
print(p1._Person__age)  # 访问私有属性
```

### 私有方法

```py
class Person:
    def __init__(self,name,age):
        self.name = name 
        self.__age = age

    def __private_method(self):
        print('这是一个私有方法')

p1 = Person('张三',20)
p1._Person__private_method()  # 调用私有方法
```

### property

只读属性

@property 可以将一个方法转换为属性来调用

```py
class Person:
    def __init__(self,name,age):
        self.name = name
        self.__age = age    
    @property
    def age(self):
        if self.__age > 18:
            return 18
        return self.__age
p1 = Person('张三',20)
print(p1.age)  # 20
```

读写属性：

将方法名设置为去掉双下划线的私有属性名，使用`@属性名.setter`装饰

```py
class Person:
    def __init__(self,name,age):
        self.name = name
        self.__age = age    
    @property
    def age(self):
        if self.__age > 18:
            return 18
        return self.__age
    @age.setter
    def age(self,age):
        self.__age = age
p1 = Person('张三',20)
print(p1.age)  # 20
p1.age = 15
print(p1.age)  # 15
```

装饰的方法不要和变量重名

## 方法的重写

#### 在一个类中 后面定义的方法会将前面定义的方法覆盖掉

在子类中,可以定义和父类方法同名的方法 (方法重写)，调用的时候会执行子类中重写的方法

```py
class Person:
    def eat(self,a):
        print('person eat')

class Student(Person):
    def eat(self,a,b):
        super().eat(a)
        print('student eat')

s= Student()
s.eat(10,20)
```

## 方法解析顺序

可以通过`类名.__mro__`查看当前类方法的解析顺序

- 顺序和定义子类的时候的继承顺序有关

- 当前类->继承的父类1->继承的父类2->父类的父类

```py
class A:
    def __init__(self,a,b,c):
        self.a = a
        self.b = b
        self.c = c

class B:
    def __init__(self,d,e,f):
        self.d = d
        self.e = e
        self.f = f

class C(A,B):
    def __init__(self,a,b,c,d,e,f):
        A.__init__(self,a,b,c)
        B.__init__(self,d,e,f) 
        
```

也可以用

```py
        #A.__init__(self,a,b,c)
        #B.__init__(self,d,e,f) 
        super().__init__(a,b,c)
        super(A,self).__init__(d,e,f)
```

或者

```py
class Parent1:
    def __init__(self,value1,**kwargs):
        print('Parent1')
        super().__init__(**kwargs)
        self.value1 = value1

class Paren2:
    def __init__(self,value2,**kwargs):
        print('parent2')
        super().__init__(**kwargs)
        self.value2 = value2
class Child(Paren2,Parent1):
    def __init__(self, value2, value1):
        super().__init__(value2=value2,value1=value1)

c = Child(1,2)
print(c.value1)
print(c.value2)
```

## 多态

一个对象的多种不同形态

```py
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    def eat(self):
        print(f'{self.age}岁的{self.name}去吃饭')

class Teacher(Person):
    pass


class Student(Person):
    pass

#方法的参数 体现多态
def  eating(Person):
    Person.eat()

zs = Teacher('zs',20)
ls = Student('ls',18)
eating(zs)
eating(ls)
# 方法的返回值  体现多态
def new_obj(num):
    if num==1:
        obj = Teacher('zs',20)
    else:
        obj = Student('ls',20)
    return obj
obj=new_obj(1)
print(obj.name)
```
