# 三大特性

## 封装

将变量和函数写入类中的操作 即为封装，类中封装了属性和方法

通过封装，可以将一些细节隐藏起来，之暴露出必要的接口给使用者使用

### 私有化

#### 1.单下划线：非公开API

不具备强制力

#### 2.双下划线，名称改写

至多一个下划线结束

只能通过`_类名__x`访问

### 私有属性

```py
class Person:
    def __init__(self,name,age):
        self.name = name 
        self.__age = age

p1 = Person('张三',20)
print(p1._Person__age)  # 访问私有属性
```

### 私有方法

```py
class Person:
    def __init__(self,name,age):
        self.name = name 
        self.__age = age

    def __private_method(self):
        print('这是一个私有方法')

p1 = Person('张三',20)
p1._Person__private_method()  # 调用私有方法
```

### property

只读属性

@property 可以将一个方法转换为属性来调用

```py
class Person:
    def __init__(self,name,age):
        self.name = name
        self.__age = age    
    @property
    def age(self):
        if self.__age > 18:
            return 18
        return self.__age
p1 = Person('张三',20)
print(p1.age)  # 20
```

读写属性：

将方法名设置为去掉双下划线的私有属性名，使用`@属性名.setter`装饰

```py
class Person:
    def __init__(self,name,age):
        self.name = name
        self.__age = age    
    @property
    def age(self):
        if self.__age > 18:
            return 18
        return self.__age
    @age.setter
    def age(self,age):
        self.__age = age
p1 = Person('张三',20)
print(p1.age)  # 20
p1.age = 15
print(p1.age)  # 15
```

装饰的方法不要和变量重名
